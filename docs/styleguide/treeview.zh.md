## 树形菜单

树形菜单可以应用默认主题的纯色背景或[选定的颜色](index.md#colors)。

此小部件具有带各式各样颜色边框的输入框。
边框颜色默认为 __muted__ 并更改为 **primary** 或 _hover_ 上的 [选定颜色](index.md#colors)。
边框厚度将通过 _focus_ 增加。

![treeview](../assets/widget-styles/treeview.gif)

```python
# 默认的树形图样式
Treeview()

# 应用了info彩色样式的树形菜单
Treeview(bootstyle='info')
```
